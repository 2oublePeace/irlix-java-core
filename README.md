## Задача 1:
Напишите программу, которая создает экземпляр данного класса:
```java
class Counter {
    private int count;

    void increment() {
    count++;
}

    int get() {
        return count;
    }
}
```

В программе создайте три потока, каждый из которых 100000 вызывает метод increment() данного объекта. Выведите на экран итоговое значение счетчика. Исправьте класс Counter так, чтобы операция инкрементирования была атомарной.

## Задача 2:
Создайте пул из пяти потоков. В начале выполнения каждый из потоков должен приостанавливать свое выполнение в течение случайного интервала времени от 0 до 10 секунд. Далее каждый поток должен генерировать случайное число в интервале от 1 до 5. Программа должна вывести на экран строку, полученную путем конкатенации пяти сгенерированных чисел (числа также можно “склеивать” в случайном порядке).

## Задача 3:
Есть класс, описывающий некую “задачу”.Есть класс, описывающий некую “задачу”.

```java
class Task {
    private String data;

    public Task(String data) { this.data = data; }

    public Task() {}

    public String getData() { return data; }

    public void setData(String data) { this.data = data; }
}
```
Нужно создать один поток, который генерирует такие задачи (поле data заполняется случайным строковым значением). Всего нужно сгенерировать 100 задач.
Сгенерированные “задачи” нужно раздать на выполнение между двумя потоками. Выполнение задачи имитируется вызовом метода Thread.sleep() со случайным временным интервалом от 0 сек до 1 сек.
По окончании работы двух принимающих потоков нужно вывести, сколько задач выполнил каждый из потоков.
